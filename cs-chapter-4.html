Programming Concepts

This chapter covers essential programming concepts including functions, data structures, pointers, and file handling. Understanding these concepts is fundamental to becoming a proficient programmer and forms the basis for more advanced programming techniques.

4.2 Functions

Functions in Programming

A function is a block of organized, reusable code that performs a single, related action. Functions provide better modularity for applications and a high degree of code reusability.

Types of Functions:

  • Library Functions: Predefined functions available in standard libraries (e.g., printf(), scanf() in C)
  • User-defined Functions: Functions created by programmers to perform specific tasks

Function Components:

  • Function Definition: Actual body of the function containing code
  • Function Prototype: Declaration of function name, return type, and parameters
  • Function Call: Invocation of the function to execute its code
  • Return Statement: Returns value from function to calling point
Function Example

Function Prototype: int add(int a, int b);

Function Definition: int add(int a, int b) { return a + b; }

Function Call: result = add(5, 3);

Accessing Functions by Passing Values

Functions can be accessed by passing values as arguments. These values can be passed in two ways:

  • Pass by Value: A copy of the argument is passed to the function
  • Pass by Reference: The memory address of the argument is passed to the function

Concept of Storage: Automatic and External

Storage classes define the scope, visibility, and lifetime of variables and functions:

  • Automatic (auto): Default for local variables, created when function is called, destroyed when function exits
  • External (extern): Global variables visible to all program files, exists throughout program execution
  • Static (static): Preserves value between function calls
  • Register (register): Suggests compiler to store in CPU register for faster access

Concept of Recursion: Factorial and Fibonacci Problems

Recursion is a programming technique where a function calls itself to solve a problem. A recursive function must have:

  • Base Case: Stopping condition to prevent infinite recursion
  • Recursive Case: Function calls itself with modified parameters
Recursive Function Examples

Factorial: n! = n √ó (n-1)! where 0! = 1

Fibonacci: F(n) = F(n-1) + F(n-2) where F(0)=0, F(1)=1

Recursive Factorial Function

int factorial(int n) {

if (n == 0 || n == 1) // Base case

  return 1;

else // Recursive case

  return n * factorial(n-1);

}

4.3 Structures and Unions

Structure

A structure is a user-defined data type that groups related variables of different data types under a single name. Structures are used to represent records.

Structure Components

Definition: Creating a blueprint for the structure

Structure Definition

struct Student {

char name[50];

int roll_no;

float marks;

char grade;

};

Declaration: Creating variables of structure type

Structure Declaration

struct Student s1, s2;

Initialization: Assigning values to structure members

Structure Initialization

struct Student s1 = {"John Doe", 101, 85.5, 'A'};

Size of Structure: The total memory allocated to a structure is the sum of sizes of all its members, plus padding for alignment.

Accessing Member of Structure

Structure members are accessed using the dot (.) operator:

Accessing Structure Members

s1.roll_no = 102;

strcpy(s1.name, "Alice Smith");

s1.marks = 90.5;

s1.grade = 'A';

Array of Structure

An array of structures allows storing multiple records of the same type:

Array of Structures

struct Student class[60]; // Array of 60 students

class[0].roll_no = 101; // Access first student

class[0].marks = 85.5;

Union: Definition and Declaration

Union

A union is a special data type that allows storing different data types in the same memory location. Unlike structures, all members share the same memory space.

Union Definition and Declaration

union Data {

int i;

float f;

char str[20];

};

union Data data;

Difference between Union and Structure

Structure vs Union Comparison
Aspect Structure Union
Memory Allocation Each member has separate memory All members share same memory
Size Sum of sizes of all members Size of largest member
Access All members can be accessed at once Only one member can be accessed at a time
Initialization All members can be initialized Only first member can be initialized
Use Case When all members need to be stored When only one member is used at a time

4.4 Pointers

Pointer

A pointer is a variable that stores the memory address of another variable. Pointers provide direct access to memory locations and are essential for dynamic memory allocation, arrays, and functions.

Address (&) and Indirection (*) Operator

Address Operator (&): Returns the memory address of a variable

Indirection Operator (*): Accesses the value at a memory address (dereferencing)

Pointer Example

int var = 10; // Integer variable

int *ptr; // Pointer declaration

ptr = &var; // ptr stores address of var

printf("Value of var: %d\n", var); // Output: 10

printf("Address of var: %p\n", &var); // Output: memory address

printf("Value of ptr: %p\n", ptr); // Output: same address

printf("Value at ptr: %d\n", *ptr); // Output: 10

Pointer Expression and Assignment

Pointers can be used in expressions and assignments:

  • Pointer Arithmetic: Pointers can be incremented/decremented
  • Pointer Comparison: Pointers can be compared
  • Pointer Assignment: One pointer can be assigned to another
Pointer Arithmetic

int arr[5] = {10, 20, 30, 40, 50};

int *ptr = arr; // Points to first element

printf("%d\n", *ptr); // Output: 10

ptr++; // Move to next element

printf("%d\n", *ptr); // Output: 20

Call by Values and Call by Reference

Parameter Passing Methods

Call by Value: A copy of the argument is passed to the function. Changes made to the parameter don't affect the original argument.

Call by Reference: The address of the argument is passed to the function. Changes made to the parameter affect the original argument.

Call by Value vs Call by Reference

Call by Value:

void incrementByValue(int x) {

x = x + 1;

}

// In main: int num = 5; incrementByValue(num); // num remains 5

Call by Reference:

void incrementByReference(int *x) {

*x = *x + 1;

}

// In main: int num = 5; incrementByReference(&num); // num becomes 6

4.5 Working with Files

Data File

A data file is a collection of related data stored on secondary storage. Files allow permanent storage of data beyond program execution.

Concept of Data File

Files are used to store data permanently on storage devices. They provide:

  • Persistence: Data remains after program execution
  • Large Storage: Can store large amounts of data
  • Data Sharing: Multiple programs can access the same file
  • Backup: Easy backup and recovery of data

Sequential and Random File

File Access Methods

Sequential Access: Data is accessed in linear sequence from beginning to end. Must read all previous data to reach a specific location.

Random Access: Data can be accessed at any position directly. Uses functions like fseek() and ftell() to move to specific locations.

Sequential vs Random Access

Sequential: Reading a file line by line

Random: Jumping to record 50 in a database file

File Manipulation Functions

Common file operations in C programming:

  • putw(): Write integer to file
  • getw(): Read integer from file
  • putc(): Write character to file
  • getc(): Read character from file
  • fprintf(): Formatted write to file
  • fscanf(): Formatted read from file
File Operations Example

FILE *fp;

// Opening file for writing

fp = fopen("data.txt", "w");

fprintf(fp, "Hello World\n"); // Write string

putc('A', fp); // Write character

putw(100, fp); // Write integer

fclose(fp); // Close file

// Opening file for reading

fp = fopen("data.txt", "r");

char ch;

while((ch = getc(fp)) != EOF) { // Read character by character

printf("%c", ch);

}

fclose(fp);

Opening, Reading, Writing and Appending Data File

Opening: fopen() - Opens a file with specified mode

File Opening Modes

"r" - Read mode (file must exist)

"w" - Write mode (creates new or truncates existing)

"a" - Append mode (creates new or appends to existing)

"r+" - Read/Write mode (file must exist)

"w+" - Read/Write mode (creates new or truncates existing)

"a+" - Read/Append mode (creates new or appends to existing)

Reading: fscanf(), fgets(), fread() - Read data from file

Reading from File

FILE *fp = fopen("data.txt", "r");

char buffer[100];

fscanf(fp, "%s", buffer); // Formatted read

fgets(buffer, 100, fp); // Read line

int num = getw(fp); // Read integer

Writing: fprintf(), fputs(), fwrite() - Write data to file

Writing to File

FILE *fp = fopen("data.txt", "w");

fprintf(fp, "Name: %s, Age: %d\n", "John", 25); // Formatted write

fputs("Hello World\n", fp); // Write string

putw(100, fp); // Write integer

Appending: fopen() with "a" mode - Add data to end of file

Appending to File

FILE *fp = fopen("data.txt", "a"); // Append mode

fprintf(fp, "New data added at the end\n");

fclose(fp);

Closing: fclose() - Closes an opened file. Always close files after operations to free resources.

Multiple Choice Questions

1. Which storage class preserves variable value between function calls?
  • A) auto
  • B) extern
  • C) static
  • D) register
Answer: C) static
2. What is the base case condition for factorial recursion?
  • A) n == 0
  • B) n == 1
  • C) n == 0 || n == 1
  • D) n <= 1
Answer: C) n == 0 || n == 1
3. Which operator is used to access structure members?
  • A) . (dot)
  • B) -> (arrow)
  • C) * (asterisk)
  • D) & (ampersand)
Answer: A) . (dot)
4. What is the main difference between structure and union?
  • A) Syntax
  • B) Memory allocation
  • C) Data types
  • D) Functionality
Answer: B) Memory allocation
5. Which operator returns the memory address of a variable?
  • A) *
  • B) &
  • C) ->
  • D) .
Answer: B) &
6. In call by reference, what is passed to the function?
  • A) Copy of value
  • B) Memory address
  • C) Variable name
  • D) Data type
Answer: B) Memory address
7. Which file opening mode creates a new file or truncates an existing file?
  • A) "r"
  • B) "w"
  • C) "a"
  • D) "r+"
Answer: B) "w"
8. Which function is used to write an integer to a file?
  • A) putc()
  • B) putw()
  • C) fprintf()
  • D) fwrite()
Answer: B) putw()
