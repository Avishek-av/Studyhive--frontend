Web Technology II

Web Technology II focuses on client-side and server-side scripting languages for dynamic web development. This chapter covers JavaScript for client-side interactivity and PHP for server-side processing, along with database connectivity.

Introduction to Web Technology II

Web Technology Evolution

Web Technology has evolved from static HTML pages to dynamic web applications using client-side and server-side scripting. Modern web development combines front-end (client-side) and back-end (server-side) technologies.

Key Components of Modern Web Development:

  • Client-Side Scripting: JavaScript, jQuery - runs in user's browser
  • Server-Side Scripting: PHP, ASP.NET, Python - runs on web server
  • Database: MySQL, PostgreSQL - stores application data
  • Web Server: Apache, Nginx - serves web pages to clients
  • Protocols: HTTP/HTTPS - communication between client and server
Web Application Architecture

Client Tier: Browser with HTML, CSS, JavaScript

Presentation Tier: Web Server (Apache/Nginx)

Business Logic Tier: Server-side Scripting (PHP/Python)

Data Tier: Database (MySQL/PostgreSQL)

Client-Side vs Server-Side Scripting

Client-Side Scripting

Scripts that run on the user's web browser. They enhance user interface and provide interactive features without contacting the server.

Languages: JavaScript, jQuery, TypeScript

Advantages: Faster response, Reduced server load, Rich user experience

Limitations: Browser dependency, Security concerns, No database access

Server-Side Scripting

Scripts that run on the web server. They process requests, access databases, and generate dynamic content sent to the client.

Languages: PHP, Python, Ruby, Java, ASP.NET

Advantages: Database access, Secure, Browser independent

Limitations: Server load, Slower response, Requires server setup

JavaScript Fundamentals

JavaScript Overview

JavaScript is a lightweight, interpreted programming language used to create interactive effects within web browsers. It's a core technology of the World Wide Web alongside HTML and CSS.

JavaScript Data Types

  • Primitive Types:
    • String: Text data ("Hello", 'World')
    • Number: Integer and floating-point numbers
    • Boolean: true or false
    • undefined: Variable declared but not assigned
    • null: Intentional absence of value
    • Symbol: Unique identifier (ES6)
    • BigInt: Large integers (ES2020)
  • Object Type: Collections of key-value pairs

Variables and Operators

JavaScript Variables
// Variable Declaration
var oldWay = "Function scoped"; // Avoid in modern JS
let modernWay = "Block scoped"; // Preferred for variables
const constantValue = "Cannot be reassigned"; // For constants

// Operators
let a = 10, b = 5;

// Arithmetic Operators
let sum = a + b;      // 15
let difference = a - b; // 5
let product = a * b;   // 50
let quotient = a / b;  // 2

// Comparison Operators
let isEqual = (a == b);     // false
let isStrictEqual = (a === b); // false
let isGreater = (a > b);    // true

// Logical Operators
let andResult = (a > 5 && b < 10);  // true
let orResult = (a > 5 || b > 10);   // true
let notResult = !(a == b);          // true

Advanced JavaScript Concepts

Functions and Control Structures

JavaScript Functions & Control Flow
// Function Declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Function Expression
const multiply = function(x, y) {
    return x * y;
};

// Arrow Function (ES6)
const divide = (x, y) => x / y;

// if-else Statement
let score = 85;
if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else {
    console.log("Grade: C");
}

// switch-case Statement
let day = 3;
switch(day) {
    case 1: console.log("Monday"); break;
    case 2: console.log("Tuesday"); break;
    case 3: console.log("Wednesday"); break;
    default: console.log("Weekend");
}

// Loops
// for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while loop
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

// do-while loop
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 3);

Object-Based Programming & Event Handling

JavaScript Objects & Events
// Object Creation
let student = {
    name: "John Doe",
    age: 20,
    course: "Computer Science",
    displayInfo: function() {
        return this.name + " - " + this.course;
    }
};

// Accessing Properties
console.log(student.name);        // John Doe
console.log(student["age"]);      // 20
console.log(student.displayInfo()); // John Doe - Computer Science

// Event Handling
document.getElementById("myButton").onclick = function() {
    alert("Button clicked!");
};

// Modern Event Listener
document.getElementById("myButton").addEventListener("click", function() {
    this.style.backgroundColor = "blue";
});

// Form Validation Example
function validateForm() {
    let email = document.getElementById("email").value;
    let password = document.getElementById("password").value;
    
    if (email === "" || password === "") {
        alert("All fields are required!");
        return false;
    }
    
    if (password.length < 6) {
        alert("Password must be at least 6 characters!");
        return false;
    }
    
    return true;
}

Image, Event and Time Objects

Image Object: Manipulate images dynamically

Event Object: Handle user interactions (click, keypress, mouseover)

Date Object: Work with dates and times

Server-Side Scripting with PHP

PHP Overview

PHP (Hypertext Preprocessor) is a server-side scripting language designed for web development. It's embedded in HTML and executed on the server, generating dynamic web pages.

Introduction to PHP: Requirements

  • Hardware Requirements: Standard web server hardware
  • Software Requirements:
    • Web Server: Apache, Nginx
    • PHP Engine
    • Database: MySQL, PostgreSQL
    • Text Editor/IDE

Object-Oriented Programming with PHP

PHP supports Object-Oriented Programming (OOP) with classes, objects, inheritance, polymorphism, and encapsulation.

PHP Basics and Programming

Basic PHP Syntax

PHP Basic Syntax
<?php
// PHP code must be within  tags
echo "Hello, World!";

// Variables

$price = 19.99;

PHP Data Types

  • Scalar Types: integer, float, string, boolean
  • Compound Types: array, object, callable, iterable
  • Special Types: resource, NULL

PHP Control Structures
<?php

    echo "Number: $i
";

    echo "Count: $count
";

PHP Operators

  • Arithmetic: +, -, *, /, %, **

  • Logical: &&, ||, !
  • String: . (concatenation), .= (concatenation assignment)

Database Connectivity with PHP

Database Connectivity

PHP can connect to various databases to store, retrieve, and manipulate data. MySQL is commonly used with PHP for web applications.

Connecting PHP to MySQL Database

PHP MySQL Connection
<?php
// Database connection parameters
$servername = "localhost";
$username = "root";
$password = "";
$database = "school_db";

// Create connection (MySQLi Procedural)
$conn = mysqli_connect($servername, $username, $password, $database);

// Check connection
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
echo "Connected successfully";

// Create database
$sql = "CREATE DATABASE IF NOT EXISTS school_db";
if (mysqli_query($conn, $sql)) {
    echo "Database created successfully";
} else {
    echo "Error creating database: " . mysqli_error($conn);
}

// Create table
$sql = "CREATE TABLE IF NOT EXISTS students (
    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    email VARCHAR(100),
    age INT(3),
    course VARCHAR(50),
    reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)";

if (mysqli_query($conn, $sql)) {
    echo "Table students created successfully";
} else {
    echo "Error creating table: " . mysqli_error($conn);
}

// Insert data
$sql = "INSERT INTO students (name, email, age, course) 
        VALUES ('John Doe', 'john@example.com', 20, 'Computer Science')";

if (mysqli_query($conn, $sql)) {
    echo "New record created successfully";
} else {
    echo "Error: " . $sql . "<br>" . mysqli_error($conn);
}

// Fetch data
$sql = "SELECT id, name, email, course FROM students";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    while($row = mysqli_fetch_assoc($result)) {
        echo "ID: " . $row["id"] . " - Name: " . $row["name"] . 
             " - Email: " . $row["email"] . "<br>";
    }
} else {
    echo "0 results";
}

// Update data
$sql = "UPDATE students SET age=21 WHERE name='John Doe'";
if (mysqli_query($conn, $sql)) {
    echo "Record updated successfully";
}

// Delete data
$sql = "DELETE FROM students WHERE id=1";
if (mysqli_query($conn, $sql)) {
    echo "Record deleted successfully";
}

// Close connection
mysqli_close($conn);
?>

Database Operations Summary

  • CRUD Operations: Create, Read, Update, Delete
  • Connection: mysqli_connect() or PDO
  • Query Execution: mysqli_query()
  • Data Fetching: mysqli_fetch_assoc(), mysqli_fetch_array()
  • Error Handling: mysqli_error(), mysqli_connect_error()
  • Security: Prepared statements to prevent SQL injection

Multiple Choice Questions

1. Which JavaScript keyword is used to declare a block-scoped variable?
  • A) var
  • B) let
  • C) const
  • D) variable
Answer: B) let
2. Which PHP function is used to connect to a MySQL database?
  • A) mysql_connect()
  • B) connect_mysql()
  • C) mysqli_connect()
  • D) db_connect()
Answer: C) mysqli_connect()
3. Which JavaScript loop executes at least once regardless of condition?
  • A) for loop
  • B) while loop
  • C) do-while loop
  • D) forEach loop
Answer: C) do-while loop
4. In PHP, which symbol is used to declare a variable?
  • A) @
  • B) #
  • C) $
  • D) &
Answer: C) $
