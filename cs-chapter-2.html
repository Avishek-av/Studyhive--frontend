Computer Networks

A computer network is a collection of interconnected computers and devices that can communicate with each other and share resources. Networks enable data exchange, resource sharing, and communication between devices over various distances.

Introduction to Computer Networks

Computer Network

A computer network is a system of interconnected computers and peripheral devices that can communicate with each other and share resources such as data, applications, and hardware devices.

Purpose of Computer Networks:

  • Resource Sharing: Share hardware (printers, scanners), software, and data
  • Communication: Email, instant messaging, video conferencing
  • Data Sharing: Centralized databases and file sharing
  • Backup and Recovery: Centralized backup systems
  • Cost Efficiency: Reduce hardware and software costs
  • Reliability: Redundant systems and alternative data paths
Real-World Examples

1. Office Network: Computers connected to share files, printers, and internet

2. University Campus Network: Connects departments, libraries, and hostels

3. Internet: Global network connecting millions of devices worldwide

4. Home Network: Connects computers, smartphones, smart TVs, and IoT devices

Basic Elements of Communication System

A communication system consists of several key components that work together to transmit information from a source to a destination.

Communication System Model
  • Sender/Source: Device that generates data to be transmitted
  • Transmitter: Converts data into transmittable signals
  • Transmission Medium: Physical path for signal transmission
  • Receiver: Device that captures the incoming signal
  • Destination: Final device that receives the data
  • Protocols: Rules governing data communication

Elements of Data Communication Transmission

  • Message: Information to be communicated
  • Protocol: Set of rules for communication
  • Bandwidth: Maximum data transfer rate
  • Latency: Time delay in transmission
  • Throughput: Actual data transfer rate achieved
  • Jitter: Variation in packet arrival time

Communication Modes

Communication modes define the direction of signal flow between two connected devices.

Simplex Mode

Communication is unidirectional. Only one device can transmit, the other can only receive.

Examples: Keyboard to computer, Television broadcasting, Radio broadcasting

Half-Duplex Mode

Communication is bidirectional, but not simultaneously. Devices take turns transmitting and receiving.

Examples: Walkie-talkies, CB radios, Traditional two-way radios

Full-Duplex Mode

Communication is bidirectional and simultaneous. Both devices can transmit and receive at the same time.

Examples: Telephone conversation, Video conferencing, Modern networking

Local Area Network (LAN) & Wide Area Network (WAN)

Local Area Network (LAN)

A LAN covers a small geographical area such as a home, office, or building. It is privately owned and has high data transfer rates.

Characteristics: High speed (10 Mbps to 10 Gbps), Low cost, Limited geographical area, Full-time connectivity

Wide Area Network (WAN)

A WAN covers a large geographical area such as a country, continent, or the entire world. It connects multiple LANs together.

Characteristics: Lower speed compared to LAN, Higher cost, Large geographical coverage, Uses public networks

LAN vs WAN Comparison

LAN: Single office network, School campus, Home network

WAN: Internet, Banking networks, Multi-national company networks

Transmission Media & Impairments

Transmission Media Types

Guided Media (Wired)
  • Twisted Pair Cable: UTP and STP cables used in Ethernet networks
  • Coaxial Cable: Used in cable TV and broadband internet
  • Fiber Optic Cable: Uses light signals, high bandwidth, long distance
Unguided Media (Wireless)
  • Radio Waves: Bluetooth, Wi-Fi, cellular networks
  • Microwaves: Satellite communication, long-distance telephony
  • Infrared: Short-range communication, remote controls

Transmission Impairments

Common Impairments
  • Attenuation: Loss of signal strength over distance
  • Distortion: Change in signal shape
  • Noise: Unwanted signals mixed with original signal
  • Echo: Reflection of signal causing interference
  • Crosstalk: Interference from adjacent wires
  • Jitter: Irregular pulse timing in digital signals

Network Architecture

Client-Server Architecture

Centralized architecture where servers provide resources and services to multiple client computers.

Characteristics: Centralized management, Scalable, Secure, Requires server administration

Examples: Web servers, Email servers, Database servers

Peer-to-Peer Architecture

Decentralized architecture where all computers have equal capabilities and responsibilities.

Characteristics: No central server, Easy to setup, Limited security, Suitable for small networks

Examples: File sharing networks, Small office networks, Home networks

Network Terminology

  • IP Address: Unique numerical label assigned to each device
  • Subnet Mask: Separates IP address into network and host portions
  • Gateway: Device that connects different networks
  • MAC Address: Physical address of network interface card
  • DNS: Domain Name System - translates domain names to IP addresses
  • Bandwidth: Maximum data transfer rate of a network

Network Connecting Devices

Network Interface Card (NIC)

Hardware component that connects a computer to a network. Every NIC has a unique MAC address.

Modem

Modulator-Demodulator. Converts digital signals to analog for transmission over telephone lines and vice versa.

Hub

Basic networking device that connects multiple devices in a LAN. Broadcasts data to all connected devices.

Switch

Intelligent device that connects devices in a LAN. Forwards data only to the specific destination device.

Router

Device that connects different networks together. Routes data packets between networks using IP addresses.

Network Topologies

Network topology refers to the physical or logical arrangement of nodes in a network.

Bus Topology

All devices are connected to a single central cable (backbone). Data travels in both directions.

Advantages: Simple, Low cost, Easy to install

Disadvantages: Single point of failure, Difficult to troubleshoot, Limited cable length

Ring Topology

Each device is connected to two other devices, forming a circular path for signals.

Advantages: No collisions, Equal network access

Disadvantages: Single point of failure, Difficult to add/remove devices

Star Topology

All devices are connected to a central device (hub or switch). Most common topology.

Advantages: Easy to install and manage, Failure of one device doesn't affect others

Disadvantages: Central point failure, Requires more cable

Basic Concept of QoS (Quality of Service)

QoS refers to the ability to provide different priority to different applications, users, or data flows to guarantee a certain level of performance.

Network Protocol

A set of rules governing data communication between devices. Common protocols include TCP/IP, HTTP, FTP, SMTP, and DNS.

Multiple Choice Questions

1. Which device converts digital signals to analog for transmission over telephone lines?
  • A) Router
  • B) Switch
  • C) Modem
  • D) Hub
Answer: C) Modem
2. Which topology has all devices connected to a central device?
  • A) Bus
  • B) Ring
  • C) Star
  • D) Mesh
Answer: C) Star
3. What is the unique physical address of a network interface card called?
  • A) IP Address
  • B) DNS Address
  • C) MAC Address
  • D) Gateway Address
Answer: C) MAC Address
4. Which communication mode allows simultaneous two-way communication?
  • A) Simplex
  • B) Half-Duplex
  • C) Full-Duplex
  • D) Semi-Duplex
Answer: C) Full-Duplex
